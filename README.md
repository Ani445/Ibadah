# Ibadah
Ibadah is a web application for practicing Muslims to keep track of their ibadah.
It was made as the Lab project for CSE 4510: Software Development and Object-Oriented Design Lab.

##  Technologies Used
### Frontend

- EJS as view engine
- JQuery
- Aladhan API

### Backend

- Node.js
- Express.js
- MySQL
- axios
